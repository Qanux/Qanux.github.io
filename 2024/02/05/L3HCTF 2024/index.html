misc

checkin

这道题一开始给出了一题网址，打开发现什么都没有，于是去除后缀后进入到另一个页面，随便输入和flag有关的句子（或许无关也行）即可获得flag

End_of_Programming

题目的意思是通过命令远程的gpt让他生成可以AC题目的代码，且题目给出了部分底层源码
根据题目给的源码（backend\src\end.py），可以知道进行了这些处理：
第一步，检查输入不包含代码，通过Is there any code or pseudocode in the following text? Answer with ‘yes’ or ‘no’ only.与输入拼接后询问ChatGPT，判断返回结果是否和”yes”完全相同来实现；
第二步：截取输入的前2048个字符询问ChatGPT，从返回结果中找到第一个匹配cpp([\s\S]+)的片段；
第三步：将代码传到评测系统评测。
通过对源码的阅读可以发现，题目是通过gpt是否输出markdown格式的cpp代码块来判断这个是不是gpt所回答的代码。所以我们可以自己手动将算法题给做出来，然后将解题代码发给gpt，叫他以markdown的格式输出该代码块即可
至于怎么绕过代码检测，其实你在代码后面加上你要让gpt做的事情，gpt会直接按照你最后提出的要求进行操作，约等于直接无视一开始的问题

exp

reserve

ez_rust

当时做这一道题挺懵逼的，感觉题目和flag都没有什么关系，于是在程序运行时将程序的内存dump了下来，拖进010 editor搜索flag字符串，结果发现了flag的关键加密代码段，直接gpt求解

crypto

babySPN

一开始看这个加密很复杂，结果flag是对k的一个哈希值，而且k都已经给出来了，直接求解即可

exp

babySPN revenge

这题则是在原来的基础上把k给删了让你来求k。一开始完全没有思路，后面想了想，k一共只有2^32种可能，那我岂不是可以尝试将他爆破出来？
利用tqdm计算了一下，爆破28位需要4个小时，所以我们可以将这个32位的k分成16个线程来进行爆破，经历了好几个小时的等待，最终成功将k给爆破出来
